## ğŸ‘¤ 2. PANDUAN ANGGOTA (MEMBER AREA)

### 2.1 Dashboard & Informasi
- **Dashboard:** Ringkasan Saldo Simpanan, Sisa Pinjaman, dan Limit Belanja.
- **Pengumuman:** Baca info terbaru di banner dashboard.
- **Notifikasi Personal:** Cek lonceng untuk status pengajuan atau tagihan.

### 2.2 Keuangan Simpanan
- **Cek Saldo:** Menu `Keuangan > Simpanan`. Lihat saldo Pokok, Wajib, Sukarela.
- **Lihat Mutasi:** Klik salah satu kartu simpanan untuk melihat detail debit/kredit.
- **Tarik Dana:**
  1. Menu `Simpanan Sukarela`, klik **Tarik Saldo**.
  2. Input nominal dan rekening tujuan.
  3. Submit. Status akan "Pending Approval".

### 2.3 Pinjaman (Loans)
- **Simulasi Pinjaman:**
  1. Menu `Pinjaman > Simulasi`.
  2. Input Nominal, Tenor, Jenis Bunga (Flat/Anuitas/Efektif).
  3. Klik **Hitung** untuk lihat tabel angsuran.
- **Ajukan Pinjaman:**
  1. Menu `Pinjaman > Ajukan Baru`.
  2. Isi form, upload Slip Gaji.
  3. Tanda tangan digital pada layar.
  4. Kirim.
- **Bayar Angsuran (Self-Service):**
  1. Menu `Pinjaman > Tagihan`.
  2. Pilih tagihan bulan berjalan.
  3. Bayar via Transfer Virtual Account / Potong Saldo Simpanan.

### 2.4 Belanja (Mart Online)
- **Katalog & Cari Produk:** Menu `Belanja`. Filter by Kategori/Harga.
- **Keranjang & Checkout:** Pilih barang > Masuk Keranjang > Checkout.
- **Metode Bayar:** Pilih Potong Saldo, Kredit (Potong Gaji), atau Transfer.
- **Riwayat Pesanan:** Menu `Pesanan Saya`. Lacak status (Dikemas/Siap Ambil/Selesai).
- **Download Struk:** Klik ikon PDF pada detail pesanan selesai.

### 2.5 Partisipasi & Informasi
- **Voting Online:**
  1. Menu `E-Voting`. Pilih kandidat saat periode pemilihan.
  2. Klik **Vote**. Pilihan bersifat rahasia dan anonim.
- **Aspirasi/Saran:**
  1. Menu `Layanan > Kotak Saran`.
  2. Tulis kritik/saran Anda.
  3. Cek balasan pengurus pada tiket yang sama.
- **Download Dokumen (AD/ART):**
  1. Menu `Informasi > Dokumen`.
  2. Unduh PDF AD/ART, SOP, dan Peraturan Koperasi.
- **Dokumentasi Sistem (Help):**
  1. Menu `Bantuan > Panduan`.
  2. Akses User Manual Online dan Video Tutorial.
- **UAT & Lapor Bug:**
  1. Menu `Bantuan > Lapor Masalah`.
  2. Jelaskan kendala/bug yang ditemukan + screenshot.
- **Update Profil:** Menu `Profil`. Update No HP, Alamat, Foto Profil.

## ğŸ‘” 3. PANDUAN PENGURUS (MANAGEMENT)

### 3.1 Dashboard Kepengurusan
- Statistik khusus: Kas Bank, NPL (Kredit Macet), Profit Loss harian, dan Jumlah Anggota Aktif.
- Shortcut ke **16 Buku Wajib** administrasi koperasi.

### 3.2 Manajemen Keanggotaan
- **Verifikasi Anggota Baru:** Menu `Anggota > Pending`. Review data > Approve/Reject.
- **Database Anggota:** Menu `Anggota > Daftar`. Cari, View Detail, atau Non-aktifkan anggota.
- **Cetak Member Card:** Buka profil anggota > Klik **Print Card** (PDF dengan QR).

### 3.3 Pinjaman (Lending Management)
- **Approval Berjenjang:**
  1. Menu `Pinjaman > Approval`.
  2. Review Analisa Kredit (5C) & Sisa Gaji.
  3. Approval Tingkat 1 (Pengurus) -> Tingkat 2 (Ketua) jika nominal besar.
- **Pencairan Dana:** Menu `Pinjaman > Siap Cair`. Klik **Cairkan** (Transfer/Tunai).
- **Input Angsuran Manual:** Menu `Pinjaman > Bayar Manual`. Input pembayaran cash anggota di kantor.

### 3.4 Organisasi & Konten
- **Buat Pengumuman:**
  1. Menu `Informasi > Pengumuman`.
  2. Tambah Baru > Tulis konten > Publish. Notifikasi terkirim ke member.
- **Buat Event Voting:**
  1. Menu `Organisasi > E-Voting > Setup`.
  2. Buat Event > Tambah Kandidat (Foto & Visi Misi) > Set Periode Aktif.
- **Balas Aspirasi:**
  1. Menu `Organisasi > Aspirasi Masuk`.
  2. Baca saran > Klik **Balas/Tindak Lanjuti**.
- **Dokumen Tata Kelola (SOP):** Menu `Organisasi > Dokumen`. Upload file PDF SOP/Peraturan baru.

### 3.5 Administrasi Kantor (Sekretaris)
- **Inventaris Aset:**
  1. Menu `Organisasi > Aset`. Tambah data aset (Laptop/Meja/Gedung).
  2. Update kondisi aset (Baik/Rusak) & nilai penyusutan.
- **Daftar Pengurus:** Menu `Organisasi > Struktur`. Update nama pejabat & periode jabatan.
- **Notulen Rapat:** Menu `Organisasi > Notulen`. Catat hasil rapat & upload dokumentasi.
- **Surat Menyurat:**
  1. Menu `Administrasi > Buat Surat`.
  2. Pilih Template (SK/Undangan).
  3. Isi variabel surat.
  4. Generate PDF (auto nomor surat & TTD QR).

### 3.6 Keuangan & Akuntansi (Bendahara)
- **Monitoring Jurnal:** Menu `Akuntansi > Buku Besar`. Review auto-journal dari sistem.
- **Input Jurnal Manual:** Menu `Akuntansi > Jurnal Umum`. Catat transaksi penyesuaian/biaya tak terduga.
- **Rekonsiliasi Bank:** Menu `Akuntansi > Rekonsiliasi`. Upload mutasi bank > Match dengan sistem.
- **Proses SHU Akhir Tahun:**
  1. Menu `Laporan > Proses SHU`.
  2. Input % Jasa Modal & Jasa Anggota sesuai RAT.
  3. Klik **Kalkulasi** > **Distribusi** ke saldo simpanan anggota.
- **Laporan Lengkap:** Menu `Laporan > Pusat Laporan`. Akses Neraca, Laba Rugi, Arus Kas, Perubahan Modal.

## ğŸª 4. PANDUAN MANAJER TOKO & KASIR (MART)

### 4.1 Point of Sale (Kasir)
- **Buka Shift:** Input modal awal laci.
- **Transaksi Penjualan:**
  1. Scan Barang.
  2. **Scan Member QR** (Wajib jika kredit/poin).
  3. Pilih Bayar (Tunai/Non-Tunai).
  4. Print Struk / Kirim Struk WA.
- **Tutup Shift:** Hitung uang fisik, input ke sistem, cetak laporan setoran.

### 4.2 Inventory Management
- **Master Produk:**
  1. Menu `Produk > List`. Tambah Baru / Edit.
  2. Set Harga Jual Umum vs Member. Upload Foto.
- **Kategori Produk:** Menu `Produk > Kategori`. Tambah/Edit kategori (Sembako, ATK, dll).
- **Stock Opname:**
  1. Menu `Inventory > Opname`.
  2. Pilih Rak/Kategori.
  3. Input stok fisik.
  4. Sistem hitung selisih. **Approve Adjustment** untuk update stok.
- **Kartu Stok:** Buka detail produk > Tab `Kartu Stok` untuk audit log keluar-masuk barang.

### 4.3 Pembelian (Procurement)
- **Manajemen Supplier:** Menu `Pembelian > Supplier`. Data kontak vendor & Term of Payment.
- **Purchase Order (PO):** Menu `Pembelian > Purchase Order`. Buat pesanan ke supplier.
- **Penerimaan Barang:** Buka PO > **Receive**. Input qty diterima & tgl kadaluarsa (batch).

### 4.4 Laporan & Keuangan Toko
- **Biaya Operasional Toko:** Menu `Keuangan Toko > Biaya`. Input bayar listrik/plastik/gaji part-time.
- **Setup Kategori Biaya:** Menu `Keuangan Toko > Kategori`. Setup pos biaya (misal: Biaya Kebersihan).
- **Riwayat Penjualan:**
  1. Menu `Laporan > Riwayat Transaksi`.
  2. Filter per Tanggal/Kasir.
  3. Klik Detail untuk melihat item yg dibeli atau melakukan **Void** (Pembatalan).
- **Laporan Kredit (Potong Gaji):**
  1. Menu `Laporan > Piutang Anggota`.
  2. Filter periode bulan ini.
  3. Export Excel untuk diserahkan ke HRD Payroll.

### 4.5 Marketing Store
- **Voucher Promo:** Menu `Marketing > Voucher`. Buat kode diskon/potongan harga & set kuota.
- **Konsinyasi (Titip Jual):** Menu `Produk > Konsinyasi`. Terima barang titipan > Settlement penjualan dengan pemilik.

## â“ 6. FAQ (PERTANYAAN UMUM)

**Q: Bagaimana cara reset password anggota masal?**
A: Admin bisa reset via menu User Management, namun disarankan anggota menggunakan fitur "Lupa Password" di halaman login.

**Q: Apakah SHU dihitung real-time?**
A: Estimasi SHU di dashboard adalah *proyeksi*. Angka final dihitung saat Tutup Buku Akhir Tahun oleh Bendahara.

**Q: Transaksi kasir salah input, bagaimana batalkan?**
A: Transaksi yang sudah "Selesai" hanya bisa dibatalkan (Void) oleh user level Supervisor/Manager Toko via menu Riwayat Penjualan.
